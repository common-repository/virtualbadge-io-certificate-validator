This plugin helps implementing the Virtualbadge.io certificate validator into a WordPress website.

== Description ==

### ------------- English -----------------

This plugin provided by Virtualbadge.io helps the customers of Virtualbadge.io to implement the certificate validator into their own WordPress website in just a few simple steps. 
In general, certificates issued by Virtualbadge.io will also be verified on the Virtualbadge.io website, but there is an option to implement this certificate validator into the customer's own website and this plugin simplyfies the whole process for WordPress websites.
The plugin is relying directly on Virtualbadge.io as a third party party provider. This third party connection is necessary and only used for the valalidation process since a certificate can only be validated if it is also stored in the Virtualbadge.io database.
You will find Virtualbadge.io directly and its Privacy Policy and Terms of Use under the following links:
Virtualbade.io: https://www.virtualbadge.io/
Privacy Policy: https://www.virtualbadge.io/privacy-policy
Terms of Use: https://www.virtualbadge.io/terms

This plugin is only useful to people with an active Virtualbadge.io subscription.


### ------------- Deutsch -----------------

== Frequently Asked Questions ==

### ------------- English -----------------

How do I get an identifier?

- You can get the identifier in our plans on [Virtualbadge.io](https://www.virtualbadge.io) with the 'certificate validator' feature.

Can I use the 'certificate validator' plugin without a plan?

- No. An active account and the necessary feature inside the selected plan is required for this plugin.

Who are the targeted users of this plugin?

- The plugin is targeted at new and existing customers of our service to provide a simple solution for embedding the certificate validator.

Who is Virtualbadge.io?

- Virtualbadge.io is a web solution to create, manage, issue and organize certificates.

I am interested in your service. How can I become a Virtualbadge.io customer?

- You can register through our website or send us an email to [help@virtualbadge.io](mailto:help@virtualbadge.io).


### ------------- Deutsch -----------------

Wie erhalte ich einen Identifikator?

- Sie erhalten den Identifikator im Rahmen unserer Abonnements auf [Virtualbadge.io] (https://www.virtualbadge.io) mit der Funktion "Zertifikatsvalidator".

Kann ich das 'certificate validator' Plugin ohne einen Account benutzen?

- Nein. Ein aktives Konto und die notwendige Funktion innerhalb des gewählten Abonnements ist für dieses Plugin erforderlich.

Wer sind die Zielgruppen dieses Plugins?

- Das Plugin richtet sich an neue und bestehende Kunden unseres Dienstes, um eine einfache Lösung für die Einbettung des Zertifikatsvalidators zu bieten.

Wer ist Virtualbadge.io?

- Virtualbadge.io ist eine Web-Lösung zum Erstellen, Verwalten, Herausgeben und Organisieren von Zertifikaten.

Ich bin an Ihrem Service interessiert. Wie kann ich Kunde von Virtualbadge.io werden?

- Sie können sich über unsere Website registrieren oder uns eine E-Mail an [help@virtualbadge.io](mailto:help@virtualbadge.io) schicken.
